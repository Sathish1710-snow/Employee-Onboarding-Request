# Project - 1 Employee-Onboarding-Request
Automated employee onboarding with approvals, task assignments, notifications, and status tracking using ServiceNow Service Catalog and Flow Designer.

Overview:

I recently worked on designing and implementing an Employee Onboarding Request System in ServiceNow to automate and streamline the hiring onboarding process for new employees. I developed an end-to-end onboarding workflow using Service Catalog + Flow Designer in ServiceNow.

Key Features Implemented:

 1. Service Catalog Item: “New Employee Onboarding Request”
 
 2. Dynamic form to capture employee details
    
 3. Manager-based approval workflow

 4. Automated task generation after approval
    
 5. Assignment of onboarding tasks to respective teams (IT, HR, Admin)
    
 6. Status tracking for request transparency

Workflow Flow:

Employee Request → Manager Approval → HR Validation → Task Creation → Fulfillment → Completion Notification

Tools & Technologies Used:

1. ServiceNow Service Catalog

2. Flow Designer

3. ITSM Workflow concepts

4. Business Rules 

5. Notifications & Email scripting

Business Impact:

1. Reduced manual onboarding coordination

2. Improved visibility of onboarding status

3. Faster approval and execution process

4. Standardized onboarding workflow
