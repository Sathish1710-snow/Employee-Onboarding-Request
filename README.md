# Project - 1 Employee-Onboarding-Request
Automated employee onboarding with approvals, task assignments, notifications, and status tracking using ServiceNow Service Catalog and Flow Designer.

I recently worked on designing and implementing an Employee Onboarding Request System in ServiceNow to automate and streamline the hiring onboarding process for new employees.

I developed an end-to-end onboarding workflow using Service Catalog + Flow Designer in ServiceNow.

⚙️ Key Features Implemented:

✔ Service Catalog Item: “New Employee Onboarding Request”
✔ Dynamic form to capture employee details
✔ Manager-based approval workflow
✔ Automated task generation after approval
✔ Assignment of onboarding tasks to respective teams (IT, HR, Admin)
✔ Email notifications at each stage
✔ Status tracking for request transparency

🔄 Workflow Flow:

Employee Request → Manager Approval → HR Validation → Task Creation → Fulfillment → Completion Notification

🛠️ Tools & Technologies Used:
ServiceNow Service Catalog
Flow Designer
ITSM Workflow concepts
Business Rules 
Notifications & Email scripting

📈 Business Impact:

✔ Reduced manual onboarding coordination
✔ Improved visibility of onboarding status
✔ Faster approval and execution process
✔ Standardized onboarding workflow
